# TT-FAB

Floating Action Button with AI Chat capabilities.

## Features
- Screenshot/attachment support
- Chat interface
- Username/password input forms
- Website access
- Floating action button

## Installation

```bash
npm install
npm run dev
```

## Usage

The FAB provides a floating button that opens an AI chat interface with:
- File attachments (screenshots, images, documents)
- Form inputs for credentials
- Embedded website access
- Real-time chat

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
